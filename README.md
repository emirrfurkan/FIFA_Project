# EA Sports FC 24 - Player Market Value Analysis

This project aims to analyze the EA Sports FC 24 player dataset to uncover the underlying factors that determine a player's market value. 

## Project Objectives
* Understand the distribution of player attributes.
* Clean and preprocess large-scale sports data.
* Perform Exploratory Data Analysis (EDA) to find correlations.
* Conduct statistical hypothesis testing on player valuations.

## Tools Used
* Python (Pandas, Matplotlib, Seaborn, SciPy)
* Jupyter Notebook
